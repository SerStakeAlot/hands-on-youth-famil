# Programs media

Drop the "Let's Talk About It" banner clip or photo here:

- Video: MP4, WEBM (preferred), or MOV (any uppercase/lowercase extension)
- Image: PNG, JPG, JPEG, WEBP, or SVG (uppercase/lowercase both work)

The Programs page automatically loads the first media file in this folder and displays it beside the description. Replace the asset any time and reload the site to see the update.
