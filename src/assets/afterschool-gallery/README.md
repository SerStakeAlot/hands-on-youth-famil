# Afterschool Enrichment Gallery

Drop photos for the Afterschool Enrichment Program into this folder.

**Supported formats:** PNG, JPG, JPEG, WEBP, SVG

**How it works:**
- All images placed here automatically appear on the **Gallery** page (`/gallery`).
- Files are sorted by filename — rename with a numeric prefix (e.g. `01-snack-time.jpg`, `02-tutoring.jpg`) to control order.

**Recommendations:**
- Keep each file under ~500KB for fast loading.
- Use consistent aspect ratios (4:3 or 3:2) for a clean grid.
- Use descriptive filenames (e.g. `homework-help-2025.jpg`).
- Make sure you have permission to share photos of any children pictured.
